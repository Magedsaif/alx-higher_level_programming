How to Use 1-my_list.py
=======================

This module defines ``MyList`` class that inherits from list.

Usage
=====

``MyList`` class have Public instance method ``print_sorted(self)`` that prints the list, but sorted (ascending sort).

::

    >>> MyList = __import__('1-my_list').MyList
    >>> my_list = MyList()

::

    >>> print(my_list)
    []

::

    >>> my_list.print_sorted()
    []

::

    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

::

    >>> my_list[4] = 6
    >>> print(my_list)
    [1, 4, 2, 3, 6]

::

    >>> my_list.remove(1)
    >>> my_list.print_sorted()
    [2, 3, 4, 6]

::

    >>> my_list = MyList([10, 1, 5, 2])
    >>> my_list.print_sorted()
    [1, 2, 5, 10]

::

    >>> my_list = MyList(['s', 'w', 'y', 'a'])
    >>> my_list.print_sorted()
    ['a', 's', 'w', 'y']
